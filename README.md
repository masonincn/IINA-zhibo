# 使用 IINA+ 在 IINA 播放器上观看直播
### 所需软件
* [IINA播放器](https://www.iina.io/)
* [IINA+](file01/iina+.0.4.8.dmg)
### 播放效果
* 左侧是IINA+。
* 右侧是IINA播放器。
![01](file01/01.png)

---
### 添加链接
* 点击左上角「书本」图标，在右侧鼠标右击「添加」。
* 把地址粘贴到框内，点击添加即可。⚠️右侧圆点绿色是正在直播，红色是没在直播。
* 左侧第二个图标是可以登录哔哩哔哩，直接观看更方便。
![02](file01/02.png)

### 优点
* 不需要打开多个平台网页。
* 没有乱七八糟的页面广告。
* 自动选择最高分辨率。
* 支持斗鱼、虎牙、企鹅电竞、哔哩哔哩、YouTube等。
* 支持多个直播同时打开。
* 调节某一个音量大小不影响其他直播音量。
